Décomposition du projet — Par où commencer ?

Ce projet comporte 6 modules indépendants. Chacun aura sa propre spec et son plan d'implémentation. Sélectionnez le module à concevoir en premier.

🏗 Fondation
🔐
Module 1 — Auth & Gestion des utilisateurs
Authentification Supabase, rôles (admin / employé / permanent), profils personnels, gestion des services et bureaux
Requis par tous les autres modules
⚙ Core
⏱
Module 2 — Système de pointage
4 points/jour (arrivée, midi out, midi in, départ), gestion du pot d'heures, heures supp au 1/4h, missions extérieures, récapitulatif
Dépend de Module 1
⚙ Core
📅
Module 3 — Calculateur d'horaires
Options A/B, horaire d'été, mi-temps médical, 4/5 congé parental, crédit-temps, horaires par bureau/affectation, planning personnel
Dépend de Module 1
⚙ Core
🌴
Module 4 — Gestion des congés & absences
Demandes de congé, workflow d'approbation, petit chômage, jours fériés belges, certificats médicaux, historique, soldes automatiques
Dépend de Modules 1 & 3
✅ Avancé
🗓
Module 5 — Calendriers de présences
Vue générale par bureau, par service, planning mensuel, vue publique (sans détail), vue admin (avec tout le détail)
Dépend de Modules 1, 2 & 4
✅ Avancé
⚙️
Module 6 — Portail admin complet
Tableau de bord centralisé, gestion de tous les travailleurs, rapports, exports, notifications Resend, approbations, modifications de statuts
Dépend de tous les modules
💡 Recommandation : Commencer par Module 1 (Auth) car c'est la fondation de tout le reste. Ensuite Module 2 (Pointage) qui est le besoin le plus urgent et concret. Les modules suivront dans l'ordre logique.
